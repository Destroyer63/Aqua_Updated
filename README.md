Aqua Water Supplier Mobile Application
-------------------Overview--------------------
Aqua Water Supplier is a mobile application developed as a college project to modernize and simplify water delivery services. The application provides a convenient platform for customers to order water cans, track their orders, and receive timely delivery updates. It also assists suppliers in managing orders and customer information efficiently.

-------------------Features--------------------
User Registration and Authentication
Water Can Booking and Ordering
Real-Time Order Management
Delivery Status Tracking
Customer Profile Management
Order History
Firebase Database Integration
Simple and User-Friendly Interface

--------------Technologies Used----------------
Android Studio – Mobile application development
Java/Kotlin – Application programming language
Firebase Authentication – User login and registration
Firebase Realtime Database / Firestore – Data storage and management
XML – User interface design

--------------------Purpose--------------------
The main objective of this project is to digitize the traditional water supply process by providing an efficient, reliable, and user-friendly platform for both customers and water suppliers.
